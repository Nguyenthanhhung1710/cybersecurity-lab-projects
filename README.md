# Cyber Security Lab Projects

Xin chào! Tôi là sinh viên ngành An toàn thông tin và đây là repository lưu trữ các dự án học tập và thực hành của tôi trong lĩnh vực an ninh mạng.

Repository này tập trung vào việc nghiên cứu, thực hành và thử nghiệm các kỹ thuật liên quan đến bảo mật hệ thống, phân tích log, và phân tích mã độc trong môi trường phòng lab.

## Nội dung chính

### 1. Phân tích mã độc (Malware Analysis)
Tiểu luận và thực nghiệm phân tích một số ransomware phổ biến như TeslaCrypt, GandCrab và Jigsaw.  
Nội dung bao gồm:
- Phân tích tĩnh bằng IDA Pro
- Phân tích động trong môi trường sandbox
- Quan sát hành vi bằng Process Explorer, Wireshark
- Tìm hiểu cơ chế mã hóa và hoạt động của ransomware

### 2. Phân tích log hệ thống
Thực hành phát hiện dấu hiệu tấn công thông qua log hệ thống Windows:
- Windows Event Viewer
- Sysmon
- ELK Stack

### 3. Lab tấn công và phòng thủ mạng
Xây dựng môi trường lab bằng VMware với các máy ảo Kali Linux và Windows để mô phỏng các kịch bản:
- Scanning với Nmap
- Brute-force với Hydra
- Khai thác lỗ hổng bằng Metasploit
- Kiểm thử web với Burp Suite và SQL Injection

### 4. Xử lý log bằng Python
Viết các script Python để xử lý và phân tích log nhằm hỗ trợ phát hiện các hành vi bất thường trong hệ thống.

## Mục tiêu dự án
- Nâng cao kỹ năng phân tích mã độc
- Hiểu rõ cơ chế hoạt động của các cuộc tấn công mạng
- Thực hành phát hiện và phân tích dấu hiệu tấn công từ log hệ thống
- Xây dựng nền tảng cho công việc trong lĩnh vực SOC và Malware Analysis

## Công cụ sử dụng
- Kali Linux
- VMware
- IDA Pro
- Wireshark
- Sysmon
- ELK Stack
- Python
- Nmap, Metasploit, Burp Suite
